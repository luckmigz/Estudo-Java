Aula 02                                                               #  
1)Crie e teste, em Java, um algoritmo que leia 3 (três) palavras,     #  
uma a uma, separadamente, e as apresente e tela, na mesma             #   
ordem em que foram digitadas, numa única linha, separadas             #  
por espaços.                                                          #  
                                                                      #  
2)Crie e teste, em Java, um algoritmo que leia 3 (três) palavras,     #   
uma a uma, separadamente, e as apresente e tela, na ordem             #   
contrária em que foram digitadas, numa única linha,                   #      
separadas por espaços.                                                #  
                                                                      #  
3)Crie e teste, em Java, um algoritmo que leia 1 (um) número          #  
inteiro, armazene-o em uma variável do tipo inteiro, e depois         #  
apresente em tela:                                                    #  
➢ “O valor do número digitado é: <número digitado>”.                  #  
                                                                      #
4)Crie e teste, em Java, um algoritmo que leia 3 (três) números,      #
separadamente, que representem uma data,                              #  
respectivamente, dia, mês e ano, e depois apresente a tela:           #       
➢ “A data digitada foi <DD>/<MM>/<AAAA>”                              #  
                                                                      #       
5)Crie e teste, em Java, um algoritmo que leia, no formato real,      #   
o peso e a altura do usuário, e depois apresente a tela:              #  
➢ “Seus dados digitados foram: <peso>kg e <altura>m”.                 #  
                                                                      #   
6)Crie e teste, em Java, um algoritmo que leia 10 (dez)               #  
caracteres, um a um, e imprima, de uma só vez, a palavra              #  
formada pela união deles.                                             #  
                                                                      #  
7)Crie e teste, em Java, um algoritmo que leia 4 (quatro)             #  
números inteiros, um a um, de um único dígito cada,                   #  
representando, na ordem, milhar, centena, dezena e unidade.           #  
Use o tipo caractere, para garantir que não seja digitado mais        #  
de um número por vez. Após a concatenação desses números,             #  
armazene-os em uma variável do tipo String. Apresente em              #  
tela o valor final dessa String.                                      #  
                                                              	      #

Aula 03                                                               #      
1)Crie um algoritmo para calcular o número de dias que você           #   
está vivo, com base em sua idade, que deverá ser digitada.            # 
                                                                      #  
2)Crie um algoritmo para calcular a área de um retângulo, com         #  
base nas medidas de sua base e de sua altura.                         #  
                                                                      # 
3)Crie um algoritmo que leia um número inteiro e o eleve ao           #  
quadrado usando a classe Math, onde a base e o expoente               #   
são números reais.                                                    #   
                                                                      #  
4)Crie um algoritmo capaz de ler três Strings quaisquer,              #            
digitadas por meio do console, e apresente a soma do                  #  
comprimento destas Strings.                                           #  
                                                                      #   
5)Crie um algoritmo capaz de ler dez palavras quaisquer,              #  
gravadas em um arquivo texto, e as apresente no console em            # 
ordem inversa do arquivo.                                             #  
                                                                      # 
6)Crie um algoritmo que calcule o valor de uma dívida,                # 
submetida à juros compostos:                                          #  
ValorFinal = ValorInicial * (1 + J / 100) ^ N                         #  
E seja capaz de responder, se você deve para o cartão de              #  
crédito R$ 100,00, a uma taxa de juros de 10%, quanto deverá          #  
ser pago depois de 8 meses.                                           #  
Onde:                                                                 #  
• J representa os juros (em %); e                                     # 
• N representa o número de meses.                                     #  
                                                                      #      
7)Entrar com um único número inteiro, com 5 dígitos, e                #
imprimir o algarismo correspondente à casa da dezena.                 # 
                                                                      #  
8)Entrar com um ângulo em graus e imprimir seu seno, cosseno,         #  
tangente, secante, cossecante e cotangente.                           #  
                                                                      #  
9)Entrar com um número e imprimir o seu logaritmo na base 10.         #      
                                                                      #  
10)Entrar com um número e a base em que se deseja calcular o          #  
logaritmo desse número. Após isto, calcular tal logaritmo e           #  
imprimir o resultado.                                                 # 
                                                                      #  
11)Crie um algoritmo que embaralhe mensagens fazendo o                # 
seguinte: leia três frases, separe cada uma delas ao meio.            #
Então junte nesta ordem: primeira metade da segunda,                  #  
segunda metade da terceira, segunda metade da segunda,                #  
primeira metade da primeira, primeira metade da terceira,             #  
segunda metade da primeira. Concatene então as três frases            #  
originais e imprima o resultado. Na linha de baixo, escreva a         #  
frase embaralhada e compare o resultado.                              #  
                                                                      #  
12)Entre com uma data em uma variável do tipo inteiro no              #  
formato ddmmaa e imprimir dia, mês e ano separados.                   #      
                                                                      #  
13)Entre com uma data em uma variável do tipo String no               #      
formato dd/mm/aa e imprimir dia, mês e ano separados.                 # 
                                                                      #  
14)Escrever um algoritmo que leia três números reais a, b e c,        # 
calcule e escreva o resultado da expressão:                           #          
x = 2 * ( ( a – c ) / 8 ) – b * 5.                                    #      
                                                                      #  
15)Crie um algoritmo para calcular a área de um círculo, com          #  
base no seu raio (Área = PI*raio*raio, onde PI = 3.14159). Use        # 
variáveis reais.                                                      #  
                                                                      #  
16)Ler dois números inteiros e imprimir dividendo, divisor,           #  
quociente e resto.                                                    #      
                                                                      #  
17)Entrar com um número e imprimir o número, seu quadrado e           #      
sua raiz quadrada.                                                    #     
                                                                      #  
18)Criar um algoritmo que calcule e imprime a área de um              #  
triângulo.                                                            # 
                                                                      #      
19)Criar um algoritmo que calcule e imprime a área de um              #  
losango.                                                              #  
                                                                      #      
20)Criar um programa capaz de calcular o 3º lado de um                #          
triângulo, digitados os outros dois lados e o ângulo entre eles       #    
esses dois lados.                                                     #
